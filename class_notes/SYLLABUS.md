## Course Description

This course provides a broad introduction to the foundational concepts and practical skills of artificial intelligence and machine learning. It covers data preprocessing, supervised and unsupervised learning, neural networks, convolutional neural networks, recurrent neural networks, transformers, transfer learning, and large language models. The course also introduces advanced topics such as scalable AI solution development, retrieval-augmented generation, voice AI, deployment, and responsible AI.

The practical implementation side of the course emphasizes Python-based libraries and frameworks including NumPy, pandas, scikit-learn, PyTorch, and TensorFlow.

## General Objectives

- Introduce the fundamental principles and workflow of machine learning and AI model development.
- Familiarize students with major supervised and unsupervised learning algorithms and their mathematical foundations.
- Develop skill in building, training, and evaluating deep learning models such as neural networks, CNNs, and RNNs.
- Equip students with knowledge of transformers, large language models, and transfer learning techniques.
- Provide hands-on experience with Python tools for data analysis, model development, and visualization.
- Enable students to understand scalable AI solutions, deployment, monitoring, and maintenance.
- Encourage critical thinking about ethics, bias, fairness, and responsible AI practice.

## Unit I: Basics of Python

This unit builds foundational programming ability required for AI and machine learning work. It introduces Python syntax and the basic programming structures that support later algorithmic and data-oriented tasks.

Topics:

- Introduction to Python and environment setup
- Basic syntax, variables, and data types
- Operators, expressions, and type conversion
- Conditional statements: `if`, `elif`, `else`
- Loops and control flow: `for`, `while`, `break`, `continue`
- Functions: definition, parameters, and return values
- Data structures: lists, tuples, sets, and dictionaries
- Introduction to object-oriented programming
- Classes, objects, and attributes
- Methods and constructors such as `__init__`

## Unit II: Python for Machine Learning

This unit introduces the practical libraries used for data analysis and visualization. It focuses on manipulating structured data, creating plots, and using Python as a machine learning workbench.

Topics:

- Introduction to NumPy arrays and basic operations
- Advanced NumPy: reshaping, broadcasting, and mathematical functions
- Introduction to pandas: Series and DataFrame basics
- Data manipulation with pandas: indexing, filtering, grouping
- Introduction to matplotlib: line plots, scatter plots, and histograms
- Advanced matplotlib customization and subplot usage
- Introduction to seaborn: boxplots, heatmaps, and pairplots
- Data visualization best practices
- Mini project on exploratory data analysis and visualization
- Review and discussion

## Unit III: Mathematical Foundation of AI

This unit prepares students mathematically for machine learning and deep learning. It combines linear algebra, calculus, probability, and statistics with practical AI-focused intuition.

Topics:

- Vectors, matrices, and matrix operations
- Matrix multiplication, transpose, inverse, eigenvalues, and eigenvectors
- Derivatives, chain rule, and gradients
- Partial derivatives and gradient descent intuition
- Random variables, PMF, PDF, and CDF
- Mean, variance, and standard deviation
- Bayes' theorem and conditional probability
- Common distributions such as normal, binomial, and Bernoulli
- Practical linear algebra with NumPy
- Implementing derivatives and gradients in Python
- Probability and statistics with Python libraries
- Hands-on application of mathematical ideas in AI models

## Unit IV: Supervised Learning

This unit introduces machine learning with labeled data. It emphasizes implementation and evaluation of core supervised models.

Topics:

- The machine learning workflow: preprocessing, training, and evaluation
- K-Nearest Neighbors implementation
- Naive Bayes classifier implementation
- Decision Tree implementation
- Key metrics for evaluation

The unit is designed to connect data preparation with predictive modeling and performance measurement.

## Unit V: Unsupervised Learning

This unit focuses on learning structure from unlabeled data. It introduces clustering and dimensionality reduction as tools for discovering patterns and representing data more compactly.

Topics:

- Introduction to unsupervised learning
- Clustering versus dimensionality reduction
- K-Means clustering algorithm
- Hierarchical clustering
- Principal Component Analysis
- t-SNE and visualization techniques
- Customer segmentation project using clustering
- Key metrics

## Unit VI: Deep Learning (1)

This unit introduces artificial neural networks and the foundations of deep learning. It combines conceptual understanding with practical implementation.

Topics:

- What is an artificial neural network
- Biological inspiration and basic terminology
- Components of a neuron: inputs, weights, and bias
- Activation functions: Sigmoid, ReLU, and Tanh
- Forward propagation
- Loss functions: MSE and cross-entropy
- Backpropagation and its mathematics
- Gradient descent optimization
- Optimizers: Adam, RMSprop, and SGD
- Regularization: Dropout and L2 regularization
- Neural networks from scratch using NumPy
- Training neural networks on sample datasets

## Unit VII: Convolutional Neural Networks and Transfer Learning

This unit focuses on deep learning for image-related tasks and the reuse of pretrained models.

Topics:

- CNN architecture: convolution, pooling, fully connected layers
- Understanding padding and stride
- Popular CNN architecture overview
- Transfer learning and fine-tuning CNNs
- Practical image classification project using CNNs
- Visualization and interpretation of CNN models

## Unit VIII: Sequence Modeling, Transformers, and Advanced Architectures

This unit covers sequence models and modern architectures for sequential and language data.

Topics:

- Introduction to Recurrent Neural Networks
- Long Short-Term Memory networks
- Gated Recurrent Units
- Applications of RNNs in sequence modeling and time series
- Introduction to transformer architecture
- Attention mechanism
- Practical project on text generation or sentiment analysis using RNNs or transformers
- Overview of advanced architectures such as GANs and diffusion models
- Ethical considerations in deep learning and AI

## Unit IX: Large Language Models and Pre-trained Models

This unit extends transformer-based learning into large language model usage and fine-tuning.

Topics:

- Transformer architecture and attention mechanism
- Platforms and tools for pre-trained models
- Using pre-trained models for text generation and analysis
- Fine-tuning large language models for specific tasks
- Project on domain-specific chatbot development

## Unit X: Advanced LLMs and Retrieval-Augmented Generation

This unit focuses on combining large language models with external knowledge and real-world AI workflows.

Topics:

- Overview of large language models and their capabilities
- Introduction to retrieval-augmented generation
- Architecture and workflow of RAG systems
- Integrating external knowledge bases with LLMs
- Fine-tuning and customizing LLMs for domain-specific tasks
- Deploying AI models with APIs and cloud platforms
- Monitoring and maintaining AI systems in production
- Capstone project involving an end-to-end AI solution with LLMs and RAG

## Unit XI: Advanced AI Applications — Voice, TTS, and Future Trends

This unit explores speech technologies and future-facing AI applications.

Topics:

- Overview of voice AI technologies and Gemini Voice
- Fundamentals of text-to-speech systems
- Neural TTS architectures such as Tacotron and WaveNet
- Implementing Gemini Voice and related use cases
- Speech synthesis pipeline from text to natural speech
- Real-world voice AI applications
- Integration of voice AI with other modalities such as NLP and vision
- Ethical and privacy considerations in voice AI
- Future trends in voice and speech technologies
- Hands-on project for a voice-enabled AI application

## Unit XII: Building and Scaling AI Solutions

This unit introduces production-level thinking for AI systems and large-scale infrastructure.

Topics:

- Introduction to scalable AI systems
- Architecture principles for scale
- Data engineering for large-scale pipelines
- Model optimization for efficiency and performance
- Distributed training and parallelism
- Cloud deployment on AWS, GCP, and Azure
- Containerization and orchestration with Docker and Kubernetes
- Monitoring, logging, and maintenance of scalable AI systems
- Managing production AI costs and resources
- Case studies of real-world scalable AI solutions

## Unit XIII: AI Research and Innovation — Understanding the Frontier

This unit encourages students to engage with the research and innovation side of AI.

Topics:

- Introduction to AI research and innovation
- Reading and analyzing research papers
- Understanding datasets, metrics, and experimental design
- Exploring novel architectures and model trends
- Reproducing research with open-source tools
- Identifying real-world problems and framing research questions
- Rapid prototyping of innovative AI solutions
- Using APIs, pretrained models, and frameworks
- Ethics, bias, and responsible AI innovation

## Tutorials

The syllabus includes tutorial activities intended to support hands-on learning in smaller groups.

Tutorial themes:

- Overview of machine learning workflow and preprocessing
- Hands-on supervised learning with KNN, Naive Bayes, and Decision Trees
- Unsupervised learning with clustering and dimensionality reduction
- Introduction to deep learning by building simple neural networks from scratch
- Practical work on CNNs and transfer learning
- Using large language models and fine-tuning pre-trained NLP models

## Practical Works

The syllabus also defines practical work components that emphasize implementation and experimentation.

Practical themes:

- Data handling and visualization with NumPy, pandas, matplotlib, and seaborn
- Supervised learning from scratch and with scikit-learn
- Clustering and PCA projects
- Building and training basic neural networks with NumPy
- Developing CNN models for image classification
- Fine-tuning transformer-based models for domain-specific chatbot applications
- Deploying AI models and evaluating performance

## Evaluation System

The syllabus defines both external and internal evaluation.

### External Evaluation

- Semester-end examination: 50 marks

### Internal Evaluation

- Class attendance and participation: 5 marks
- Practical work and internal activities contribute to the remaining internal score
- Total internal marks: 50

### Overall Marks

- Full marks: 100
- Internal + external structure: 50 + 50

## Student Responsibilities

- Maintain at least 80 percent attendance
- Achieve at least 45 percent in internal assessments to qualify for the final exam
- Attend regular sessions and complete assignments on time
- Proactively cover missed content
- Understand that missed tests or exams are not guaranteed re-examination opportunities

## Syllabus Summary

The syllabus is designed as a full AI/ML pathway beginning with Python and mathematics, moving through classical machine learning and deep learning, and extending into transformers, large language models, production systems, voice AI, and research-oriented innovation. It combines foundational theory, hands-on implementation, and broader awareness of ethics, scalability, and future AI directions.
